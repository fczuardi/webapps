- **props** vs **state**
    - https://github.com/uberVU/react-guide/blob/master/props-vs-state.md
    - Resumidamente:
        - props são os atributos do componente, é configurado na hora de
         instanciá-lo.
        - state é uma foto do estado do componente no tempo, muda de acordo
        com eventos de UI
